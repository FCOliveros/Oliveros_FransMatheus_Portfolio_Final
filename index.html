Projects:

Basic Cards with functional Modals using Bootstrap Framework

Cards are used in multiple services but are primarily used in shopping websites to display product names product image/s, product description, and a button that leads them to the product that was being displayed in
the card. Using bootstrap, I replicated this concept with 3 different sample products that when the button in said card is clicked, a modal will appear showcasing the product's name, image, description, and a button that
showcases an alert, thanking the user for purchasing the product as a placeholder function for the button.

Cards layout replication using Tailwind Framework

In this project, I then replicated the design of the cards I created in the first project but this time using the Tailwind Framework. This project lacked the functionalities that the Bootstrap project had as
Bootstrap is primarily a component-based framework whereas Tailwind is a utility-first framework primarily focused on creating custom designs rather than pre-defined UI elements. This project's goal was to replicate
the appearance of the cards from the first project but using the Tailwind framework without the functionalities.

Barangay Registration System using C++

A barangay registration system is a system used by communities in the Philippines called "Barangays" that allows them to maintain records of the people in the community such as population count, population demography,
and the ability to report incidentsas well as register local business within the barangay. The system also features a simple security system to access the features via a registration and login feature where login credentials are
saved in a text file that is saved in the barangay's computer as it is only a basic security system. The system was built using C++ only therefore the system only runs using CMD or Command Prompt.
